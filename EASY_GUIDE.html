AGitHub에 내 사이트 폴더 올리기
GitHub은 내 파일을 인터넷에 올려두는 "클라우드 폴더"라고 생각하면 됩니다.
1
브라우저에서 github.com 접속 → 오른쪽 위 Sign up 클릭 → 이메일, 비밀번호로 계정 만들기
이미 계정이 있다면 이 단계는 건너뛰고 Sign in으로 로그인하세요.
2
로그인 후 화면 오른쪽 위에 + 모양 아이콘 클릭 → 나오는 목록에서 New repository 클릭
화면 예시: 오른쪽 위에 동그라미(내 프로필 사진) 바로 왼쪽에 + 아이콘이 있어요.
3
Repository name 칸에 ee-studio 입력 (이름은 원하는대로 바꿔도 됩니다) → 화면 아래쪽 Public이 선택되어 있는지 확인 → 맨 아래 초록 버튼 Create repository 클릭
4
새로 만들어진 빈 페이지에서 uploading an existing file이라는 파란 글씨 링크를 클릭 (또는 Add file 버튼 → Upload files)
5
제가 드린 ee-studio 압축을 먼저 풀어두세요 (마우스 오른쪽 클릭 → 압축 풀기). 그 안에 있는 모든 파일과 폴더를 한 번에 선택해서, 화면 가운데 점선 박스 안으로 마우스로 끌어다 놓기(드래그) 하세요.
화면 예시: "Drag files here to add them to your repository" 라는 문구가 쓰인 큰 회색 점선 박스가 보일 거예요.
6
파일이 다 올라간 걸 확인했으면, 맨 아래로 스크롤 → 초록 버튼 Commit changes 클릭
"Commit"은 그냥 "저장하기" 버튼이라고 생각하면 됩니다.
B주소(URL) 하나만 바꿔 끼우기
app.js 안에 있는 "여기에 주소를 붙여넣으세요"라고 써진 부분을, 진짜 주소로 바꾸는 작업입니다. 별도 프로그램 없이 GitHub 화면에서 바로 합니다.
1
방금 올린 저장소 화면에서 assets 폴더 클릭 → 그 안의 app.js 파일 클릭
2
화면 오른쪽 위에 연필 모양 아이콘(수정하기)을 클릭
화면 예시: 파일 내용 오른쪽 위, 연필 그림 아이콘이에요. 안 보이면 화면을 조금 넓게 키워보세요.
3
화면에 나온 글자들 중에서 PASTE_YOUR_APPS_SCRIPT_WEB_APP_URL_HERE 라는 부분을 찾아서, 마우스로 그 글자만 드래그해서 선택 → 미리 복사해둔 Apps Script 주소(.../exec로 끝나는 긴 주소)를 붙여넣기(키보드 Ctrl+V, 맥은 Cmd+V)
따옴표(" ") 안의 글자만 바뀌어야 합니다. 따옴표 자체는 지우지 마세요.
예: "PASTE_YOUR_..._HERE" → "https://script.google.com/macros/s/.../exec"
4
화면 아래쪽(또는 오른쪽 위)의 초록 버튼 Commit changes 클릭 → 다시 한번 Commit changes 클릭해서 저장
C사이트 주소 만들기 (GitHub Pages 켜기)
1
저장소 맨 위 메뉴에서 Settings 클릭 (톱니바퀴 아이콘일 수도 있어요)
2
왼쪽 메뉴 목록에서 아래쪽에 있는 Pages 클릭
3
"Branch" 라고 써진 드롭다운(선택 상자)을 클릭 → main 선택 → 옆에 있는 Save 버튼 클릭
4
1~2분 기다렸다가 같은 페이지를 새로고침(F5) 하세요. 초록색 박스로 "Your site is live at..." 라는 문구와 함께 주소가 나타납니다. 그 주소가 바로 학생들에게 보낼 사이트 링크예요.
형식: https://[내아이디].github.io/ee-studio/
✓막히면 이렇게 해주세요
어느 단계에서 막혔는지 화면을 캡처(스크린샷)해서 저한테 보여주시면, 그 화면을 보고 바로 다음에 뭘 눌러야 하는지 알려드릴게요. 지금까지 해보신 것 중 어디까지 되고 어디서부터 안 되는지만 알려주셔도 충분합니다.
